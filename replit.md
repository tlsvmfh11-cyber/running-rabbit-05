# 강남 퍼펙트 SEO 최적화 랜딩 페이지

## Overview
강남 퍼펙트 업장의 SEO 최적화 랜딩 페이지. 타겟 키워드 "강남 퍼펙트"로 검색 엔진 상위 노출을 목표로 합니다.

## Architecture
- **Frontend**: React + Vite + Tailwind CSS (단일 페이지 랜딩)
- **Backend**: Express.js (robots.txt, sitemap.xml 서빙)
- **Styling**: 블랙 & 골드 테마, 다크 모드 기본

## Key Files
- `client/index.html` - SEO 메타태그, 구조화 데이터 (JSON-LD), OG/Twitter 카드
- `client/src/pages/Home.tsx` - 메인 랜딩 페이지 컴포넌트
- `client/src/index.css` - 블랙 & 골드 테마 CSS 변수
- `server/routes.ts` - robots.txt, sitemap.xml 엔드포인트
- `client/public/` - 이미지 파일 (0~6.webp)

## SEO Features
- Schema.org JSON-LD: WebSite, WebPage, LocalBusiness, Organization, FAQPage, Article, HowTo, BreadcrumbList
- Open Graph + Twitter Card 메타태그
- Semantic HTML (h1/h2/h3 계층, section, nav, article)
- robots.txt (Google, Naver Yeti, Bing 최적화)
- sitemap.xml (이미지 사이트맵 포함)
- 내부 링크 구조 최적화
- 키워드 밀도 최적화
- 모바일 최적화 CTA
